<h1> Aula 03 - Introdução às Rotas, Controllers e Views </h1>

<h3>1. Iniciando o projeto</h3>

```
composer create-project --prefer-dist laravel/laravel app_super_gestao 
php -S localhost:8000
```

- se quiser inserir uma versão para o laravel, inserir ao final dessa linha de comando, com aspas duplas, como "7.0.0".
- eu utilizei:

```
composer create-project laravel/laravel meu-projeto
cd meu-projeto
php artisan serve
```
